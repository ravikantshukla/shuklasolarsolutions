# Image Resources

Place all editable website images in this folder to avoid merge conflicts and keep asset management simple.

## Current placeholders
- `hero-pm-surya-placeholder.svg` → hero background image for PM Surya Ghar section.

## How to replace
1. Keep the same file name to avoid HTML/CSS edits, or update references in `index.html`.
2. Prefer compressed formats (`.webp`, optimized `.jpg`, lightweight `.svg`) for faster loading.
3. Recommended hero size: `1600x900`.
